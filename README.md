## About

Personal portfolio website built with an IDE-inspired dark theme (VSCode-style). It presents my professional background, featured projects, technical publications, and resume in two languages.

## Features

- **Bilingual** — Portuguese (`index.html`) and English (`index-us.html`)
- **IDE / VSCode-inspired UI** — dark theme, Fira Code font, tab navigation, grid background
- **Animated sections** — scroll animations via AOS
- **Responsive** — adapts to mobile, tablet, and desktop
- **Interactive project modals** — detailed view for featured projects
- **Resume download** — PDF available in PT-BR and EN

## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | Tailwind CSS (CDN) |
| Animations | AOS (Animate On Scroll) |
| Icons | Feather Icons |
| Font | Inter + Fira Code (Google Fonts) |
| Hosting | GitHub Pages |

## Sections

| Section | Description |

## Running Locally

No build step required — open `index.html` or `index-us.html` directly in any browser.

```bash
# Clone the repository
git clone https://github.com/igordriguess/portfolio-irodrigues.github.io.git

# Open in browser
start index.html
```
